virtual bool kyotocabinet::TreeDB::open ( const std::string &  path,
uint32_t  mode 
) [virtual]

Open a database file.

Parameters:
path the path of a database file.
mode the connection mode. HashDB::OWRITER as a writer, HashDB::OREADER as a reader. The following may be added to the writer mode by bitwise-or: HashDB::OCREATE, which means it creates a new database if the file does not exist, HashDB::OTRUNCATE, which means it creates a new database regardless if the file exists, HashDB::OAUTOTRAN, which means each updating operation is performed in implicit transaction, HashDB::OAUTOSYNC, which means each updating operation is followed by implicit synchronization with the file system. The following may be added to both of the reader mode and the writer mode by bitwise-or: HashDB::ONOLOCK, which means it opens the database file without file locking, HashDB::TRYLOCK, which means locking is performed without blocking.
Returns:
true on success, or false on failure.
